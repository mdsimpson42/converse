You can contribute to this by suggesting content and blog posts, or by helping with the development of the site.

Blog Submissions

We would like to encourage as many people, from a wide and diverse range of backgrounds, to contribute to the blog. You could share your mental health story, talk about your experiences with mental health in academia, or discuss how you have supported others. You could share tips and advice, or links to resources that you have found helpful.

When you submit a blog post, you can choose to do so anonymously or you can create an author profile. If you choose to create an author profile, you can include a short bio and a photo of yourself, as well as links to your social media profiles or website.

How to Submit

There are two ways to submit a blog post.

1. Pull Request

The site is a Jekyll app hosted on GitHub pages. You can add a blog post to the site via a pull request on the GitHub repository. Full instructions can be found on the repository's wiki.

2. Web Form

We understand that not everyone is familiar with GitHub or has an account. If you would prefer, you can submit a blog post using the form below.

Project/Community Submissions

If you know of any more projects relating to mental health in our community that you think we should include, please let us know using the form below and we will add it to the list.

As with blog posts, you can add projects/communities via Pull Request (Instructions on the repository's wiki) or using the form below.

Contributing to the Site

As you can probably tell, I am not an experienced web developer! If you have ideas for how to improve the site, particularly changes that will make it as user-friendly and accessible as possible, please get in touch. I would be very grateful for any help.

  • Check out the GitHub repo for more information.
  • We also have a Project Board, which lists all the open issues and feature requests.
